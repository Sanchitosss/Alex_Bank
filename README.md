Банковское приложение на Flask.

**Стек:** Python, Flask, SQLite, HTML, CSS

**Реализовано:**
- Регистрация и авторизация
- Валидация данных (возраст, телефон, пароль)
- Работа с БД
- Обработка GET/POST запросов

**В планах:**
- Переход на PostgreSQL
- Подключение SQLAlchemy (ORM)
- Покрытие тестами (pytest)
